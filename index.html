Agent Poro

Windows Phone application where you can view summoners profile and browse LoL content.

About App

Agent Poro is free to use application that allows you to browse League of Legends content. With Agent Poro you can check:
- Summoner statistics, leagues, last matches, rune and mastery pages
- Champions descriptions
- Items descriptions
- Skin sales
- Champion sales
- Current rotation

Also Agent Poro may notify you about new sales and rotation!
